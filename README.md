# Scanner de ports
## Description
Ce programme scanne tous les ports TCP communs de l'hôte souhaité (IP/Host).

## Utilisation
1. Le packages à télécharger :
- `pip install sys`
- `pip install json`
- `pip install socket`
- `pip install pyfiglet`

2. Lancement : `py main.py [ip/host]` 
<br>*Par exemplle : `py main.py 127.0.0.1`*
